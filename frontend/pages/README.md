# Pages

## Purpose
The pages directory contains Next.js page components that define the routes and pages of the frontend application.

## What Lives Here
- **create-position.tsx** - Page for creating new automated liquidity positions

## How It Fits Into the System

- **Depends on**: Next.js, React, Wagmi, Viem
- **Provides**: Page routes and user-facing pages for the Dexter MVP
- **Part of**: The frontend application, defining the application structure
